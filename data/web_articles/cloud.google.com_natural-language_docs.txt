Cloud Natural Language documentation
The Cloud Natural Language API provides natural language understanding
  technologies to developers, including sentiment analysis, entity analysis,
  entity sentiment analysis, content classification, and syntax analysis.
  This API is part of the larger Cloud Machine Learning API family.Learn more
For information on which languages are supported by the Natural Language,
  seeLanguage Support.
